    主题色

    乌龙茶原创大约 2 分钟WebVue主题色

    全局更改主题色有多种方式。

    关键在于使用CSS 变量open in new window，定义在伪根类:rootopen in new window下，可以全局访问，并通过改变变量的值，使全局生效。

    • 使用v-bind-cssopen in new window
    v-bind
    • 使用useCssVaropen in new window
    useCssVar
    • 使用setPropertyopen in new window

    注意

    麻了,vue-demoopen in new window使用ShadowDomopen in new window隔离，无法使用refopen in new window。

    通过直接对document操作。使用document.querySelectoropen in new window,document.getElementByIdopen in new window来获取元素 Dom。

    ShadowDom内访问外部元素，请加上window,使用window.document即可。

    setProperty
    <template>
  <div ref="el" id="el" class="box">Sample text, {{ color }}</div>
  <button @click="switchColor">{{ theme }}</button>
</template>
<script>
export default {
  setup() {
    const { ref, onMounted } = Vue;
    const el = ref(null);
    const color = ref("");
    const theme = ref("夜晚");
    onMounted(() => {
      //el.value = document.querySelector("#el").style;
      el.value = document.getElementById("el").style;
      //el.value=window.document.querySelector(".code-demo-container").shadowRoot.querySelector("#el").style;
      color.value = el.value.getPropertyValue("--color");
    });
    function switchColor() {
      if (theme.value === "白天") {
        color.value = "#7fa998";
        theme.value = "夜晚";
      } else {
        color.value = "#df8543";
        theme.value = "白天";
      }
      el.value.setProperty("--color", color.value);
    }

    return {
      theme,
      color,
      switchColor,
    };
  },
};
</script>
<style>
:root {
  --color: #7fa998;
}
.box {
  color: var(--color, #7fa998);
}
</style>

    场景

    有一个现成的框架,实现了深色模式,现在要添加自己的页面，如何让自定义页面随着深色模式开启而自动启动。

    CSS
    <template>
  <div data-theme="light" class="box" id="demo">
    {{ color }}
  </div>
  <button @click="switchTheme">{{ themeData }}</button>
</template>
<script>
export default {
  setup() {
    const { ref, onMounted } = Vue;
    const color = ref("#7fa998");
    const themeData = ref("light");
    const themeContainer = ref(null);
    onMounted(() => {
      themeContainer.value = document.querySelector("div[data-theme]");
      const t = document.querySelector("h1");
    });

    function switchTheme() {
      if (themeData.value === "light") {
        themeData.value = "dark";
      } else {
        themeData.value = "light";
      }
      window.document.documentElement.setAttribute(
        "data-theme",
        themeData.value
      );
      //因为shadowDom隔离，模拟
      themeContainer.value.setAttribute("data-theme", themeData.value);
    }

    return {
      themeData,
      color,
      switchTheme,
    };
  },
};
</script>
<style>
:root {
  --demo-color: #7fa998;
}

*[data-theme="light"] {
  --demo-color: #7fa998;
}

*[data-theme="dark"] {
  --demo-color: #df8543;
}
.box {
  display: inline-block;
  box-sizing: border-box;
  padding: 0 0.25em;
  font-weight: bold;
  font-size: 0.75rem;
  border: 1px solid #d9d9d9;
  border-radius: 0.25em;
  transition: all 0.3s;
  cursor: pointer;
  color: var(--demo-color);
}
</style>
