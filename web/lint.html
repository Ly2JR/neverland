    项目提交规范涉及以下几个包

    序号包名简介
    1ESLintopen in new window检查js问题并修复
    2prettieropen in new window格式化代码
    3stylelintopen in new window检查css问题并修复
    4huskyopen in new windowGit钩子
    5nano-stagedopen in new window轻量级的,将检查范围定位在提交的文件中,同lint-stagedopen in new window
    6commitlintopen in new window配置提交约定
    7cz-commitlintopen in new window提交时选择

    安装步骤

    以nano-staged、PNPM为例，按文档open in new window说明安装。

    注意

    • commitlint.config.js默认是UTF-16 LE编码格式，请重新保存为UTF-8编码格式
    • commitlint.config.js默认是CommonJs规范，请更改后缀名为cjs变成ESMoudle规范
    • 请自行在commitlint.config.cjs里添加rules。
    • 相关包的配置见对应官方配置说明

    1. 安装nano-staged

      pnpm add nano-staged -D

    2. 新建nano-staged的配置文件open in new window

      {
   "*.{js,ts}": "prettier --write",
   "*.css": ["stylelint", "eslint --fix"]
}

    3. 安装husky

      pnpm add husky -D

    4. 启用Git钩子

      npx husky install

    5. 添加预提交命令到钩子上

      npx husky add .husky/pre-commit "./node_modules/.bin/nano-staged"

    6. 为了每次安装自动执行，需要编辑package.json

      "scripts": {
  "postinstall": "npx husky install"
}

    7. 安装prettier、stylelint、eslint

      pnpm add prettier styleint eslint -D

    8. 安装commitlint

      pnpm add @commitlint/config-conventional @commitlint/cli -D

    9. 配置约束

      echo "module.exports = { extends: ['@commitlint/config-conventional'] };" > commitlint.config.js

    10. 添加提交约束

    npx husky add .husky/commit-msg  'npx --no -- commitlint --edit ${1}'
    1. 测试验证
    npx commitlint --from HEAD~1 --to HEAD --verbose
    1. 添加选择提交

    配置commitizen adapter

    pnpm install --save-dev @commitlint/cz-commitlint commitizen inquirer@8

    在package.json中添加

    {
   "scripts": {
      "commit": "git-cz"
   },
   "config": {
      "commitizen": {
         "path": "@commitlint/cz-commitlint"
      }
   }
}

    基本结构

| |-.husky

| |--|pre-commit
| |-src

    • .husky内容如下:
    #!/usr/bin/env sh
. "$(dirname -- "$0")/_/husky.sh"

npx --no -- commitlint --edit ${1}
    • pre-commit内容如下:
    #!/usr/bin/env sh
. "$(dirname -- "$0")/_/husky.sh"

./node_modules/.bin/nano-staged
