## 什么是设计模式

设计模式是您在实际应用中一次又一次发现的软件设计问题发展。

模式是关于对象的可重用设计和交互的。

23 四人组 （GoF） 模式通常被认为是所有其他模式。

它们分为三组：创造型、结构型和 行为学(完整列表见[此处](https://www.dofactory.com/net/design-patterns)）)
